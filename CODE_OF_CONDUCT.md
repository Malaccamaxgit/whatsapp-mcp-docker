# Code of Conduct

> **Community standards** — Respectful and inclusive environment for all contributors.

## Standards

- Be respectful and considerate
- Accept constructive feedback gracefully
- Focus on what's best for the project
- Avoid inappropriate or offensive behavior

## Scope

This applies to all project spaces including GitHub issues, pull requests, and discussions.

## Reporting

Report unacceptable behavior to benjamin.alloul@gmail.com.

Reports will be reviewed and addressed as appropriate for a solo-maintained project.

## Enforcement

| Level | Consequence |
|-------|-------------|
| **Correction** | Private warning, clarification of inappropriate behavior |
| **Warning** | Formal warning with consequences for continued behavior |
| **Temporary Ban** | Temporary ban from community interaction for specified period |
| **Permanent Ban** | Permanent ban from any public community interaction |
